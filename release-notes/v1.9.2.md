# v1.9.2 — Distance to each station

Adds the as-the-crow-flies distance (Luftlinie) from your reference point
to every station, shown beneath the map-pin. No breaking changes.

## Added

- **Distance per station (#61).** Each station now carries a `distance_m`
  attribute — the straight-line distance (metres) from your configured
  location, or from the tracked device's live position in dynamic mode.
- **Distance on the card.** The value renders as a small caption under the
  map-pin ("850 m", "12,3 km"), tucked beneath the icon so rows stay
  compact. Toggle it with the new **Show distance** switch in the card
  editor (on by default).

## Compatibility

- Home Assistant: ≥ 2025.1.0
- HACS: ≥ 2.0.0
